Prework Study Guide

HTML

  • The head element contains information about the webpage, metadata, that helps the browser to render elements of the page correctly
  • The body element represents the visible content shown to the user. All the elements that we want visible to the user and interact with must be inside the body element.
  • The content is contained between an opening tag and a closing tag
  • Elements nested inside others are child elements, and those that hole the child elements are parent elements
  • Semantic HTML Elements are the name of the element describing its content and the role is performs on a webpage.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Every Git repository starts with the main branches, and a feature branch is created to add something new to our project
  • Git is a system that allows you to create "commits" of your work
  • 'git status': checks what branch we are currently on
  • 'git checkout -b branch-name': creates a new branch and switches to it
